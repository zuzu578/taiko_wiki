# 태고의달인 위키가 뭘까

태고의달인 위키는 사실 디시인사이드 위키를 보고 영감받아서 개발 하게 된 위키이다. 사실 위키 기능은 jsp로 개발하는 태고의달인 에서도 개발했기때문에 리액트에서 개발하려고 따로 프로젝트를 딴것이다.

# 기존 태고위키와 다른점 

우선 틀 작성 기능을 만들었다. 
틀이란 무엇이냐면

# 본인 태고위키에서의 틀

이렇게 틀을 넣을수있는데 

디시위키에서 틀넣는법은 {{레전드}} 라던가 {{갓곡}} 이렇게 작성하게되면 틀을 넣을수있다. 
하지만 본인이 개발한 위키에서는 틀을 input 에다 레전드,갓곡,전설,눈물 등의 태그를 넣으면 틀을 생성해주는 util을 만들어서 그려주고 있다.
